# vue-modal-input  
  
## 概要  
コンポーネント間の値の受け渡しサンプル、reactive/refそれぞれで、コンポーネントへ渡して  
コンポーネント内で入力した内容を変数に代入するパターンを作成しました。  
  
### セットアップ  
```sh
npm install
```
  
### 実行  
```sh
npm run dev
```
